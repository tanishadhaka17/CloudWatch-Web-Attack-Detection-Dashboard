# CloudWatch Traffic - Web Attack Detection Dashboard

This project analyzes and visualizes web attack traffic logs collected from **AWS CloudWatch** to identify suspicious activity, attacker behavior, and traffic anomalies.  
The dashboard is built using **Power BI** with data preprocessing in **Excel**.
